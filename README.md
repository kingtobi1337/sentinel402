# Sentinel402

**Autonomous agents buy evidence, not subscriptions.**

Sentinel402 is a Hedera-native x402 procurement agent for pay-per-result account forensics. It discovers priced tools, chooses the smallest evidence bundle that satisfies a goal and budget, pays every selected tool on Hedera testnet, and returns a machine-readable dossier with HashScan receipts.

> Active build for the Hedera x402 bounty. Real testnet transaction evidence and the public demo URL will be added after the credentialed E2E gate.

## Why it is different

This is not a paywalled file or a single paid market-data endpoint. Sentinel402 demonstrates an autonomous buyer that compares tools, enforces a budget, makes multiple independent x402 purchases, and aggregates the paid outputs into one auditable decision.

## Status

Bootstrap architecture is frozen. Implementation and live testnet verification are in progress.
