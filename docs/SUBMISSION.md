## Project description — three sentences

Sentinel402 is an autonomous procurement agent that discovers individually priced Hedera forensic tools, applies a goal-and-budget policy, and purchases only the evidence it needs through x402. Each protected result is released only after an exact HBAR transfer settles on Hedera testnet, while the final dossier includes deterministic mainnet evidence and a HashScan receipt for every purchased tool. The live payment flight recorder turns the full `402 → sign → verify → settle → 200` exchange into an inspectable agent-to-agent commerce demo.
